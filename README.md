# astrbot_plugin_qweather

基于和风天气 API 的 AstrBot 个人天气与通勤建议插件。

## 命令

- `/天气 [城市]`：查询明日详细天气、未来 3 天简报、穿衣和生活建议。
- `/通勤`：根据当前时间自动判断输出今日/明日上班、下班天气和交通建议。
- `/天气帮助`：查看可用命令和配置提示。

## 配置

所有参数均通过 AstrBot 插件配置页面填写，插件提供 `_conf_schema.json` 用于生成配置表单。

必填配置：

- 和风天气 API Key
- 和风天气 API Host
- 默认城市
- 家地址或所在区域
- 公司地址或所在区域
- 上班通勤时间段，例如 `08:00-09:00`
- 下班通勤时间段，例如 `18:00-19:00`
- 可选通勤方式，例如 `地铁、公交、骑行、开车、打车`

## `/通勤` 智能判断表

以上班 `08:00-09:00`、下班 `18:00-19:00` 为例：

| 当前时间 | 输出内容 |
| --- | --- |
| 00:00-07:59 | 今日上班 + 今日下班 |
| 08:00-09:00 | 当前上班时段 + 今日下班 |
| 09:01-17:59 | 今日下班 |
| 18:00-19:00 | 当前下班时段 |
| 19:01-23:59 | 明日上班 + 明日下班 |

插件会先生成结构化天气/通勤数据，再调用 AstrBot 当前默认模型整理成更自然的聊天回复；模型不可用或超时时自动使用内置模板输出。
